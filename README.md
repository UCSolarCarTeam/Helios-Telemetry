# Solar Car Helios Telemetry

Front End: https://telemetry-beta.calgarysolarcar.ca/

### Start Web App (Dev Build)

Website:

```
cd app
npm react:dev
```

App:

```
cd app
npm run electron:dev
```

### Start Server (Dev build)

```
cd server
npm run dev
```

### Start Server (Production build)

```
cd server
npm run build
npm run prod
```
