# Solar Car Helios Telemetry

Front End: https://ucsolarcarteam.github.io/Helios-Telemetry/


### Start Web App (Dev Build)

Website:

```
cd app
npm run dev
```

App:

```
cd app
npm run electron:dev
```

### Start Server (Dev build)

```
cd server
npm run dev
```

### Start Server (Production build)

```
cd server
npm run build
npm run prod
```
